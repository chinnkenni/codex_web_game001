# 雾港三小时

第一版轻量 Web 试玩。目标是 15 分钟内完成一局：三小时后雾港东区封控清洗，玩家需要在路线节点之间移动，打出行动牌，收集线索和入层条件，并尽量进入地下避难层。

当前氛围方向是雾港旧城里的中式民俗恐怖：供桌、纸扎、空座、门神贴、阴纹铜钥和戏台短波都被处理成“道具 / 征兆”。页面会随警觉值从低声、注视到追踪/合围逐渐变色；顶部 ♪ 可以开启程序化氛围音，低警觉时偏风噪和低频，警觉升高后会增加脉冲、失谐和锣点。

## 本地试玩

```bash
npm run dev
```

打开 `http://127.0.0.1:4173`。

## 手机临时预览

```bash
npm run phone:preview
```

该命令会启动只监听 `127.0.0.1` 的只读预览服务，并通过 Cloudflare Quick Tunnel 生成临时 HTTPS 链接。终端会输出手机访问链接、一次性长分享码和停止方式。

手机保持小火箭开启，打开终端里的 Cloudflare 链接，输入分享码即可。详细说明见 [docs/phone-preview.md](./docs/phone-preview.md)。

## 发布到 Cloudflare Workers

Cloudflare Worker 应用 `codex-web-game001` 使用 Static Assets 承载游戏文件，并连接 GitHub 仓库 `chinnkenni/codex_web_game001`。

- 生产分支：`main`
- 构建命令：`npm run build`
- 静态资源目录：`dist`

之后只要把变更推送到 GitHub，Cloudflare Workers Builds 会自动构建和发布。项目是纯静态页面，后续不需要数据库或服务器即可部署。

## 第一版范围

- 静态 Web 游戏，无数据库、无账号。
- 每局封锁前 3 小时，约 8-12 次关键选择。
- 核心循环：选择路线节点 → 打行动牌 → 承担成功率/资源/警觉后果 → 尝试入层。
- 每次事件后会弹出“事件回执”，即时确认获得的道具/征兆、属性变化和本次检定结果。
- 每张行动牌显示耗时、成功率、需求和风险收益。
- 本机痕迹保存在当前浏览器本地，会转化为下一局的便签和遗物效果。
- 重复游玩会形成轻度 roguelite 成长，但不需要账号或数据库。

## 之后可扩展
